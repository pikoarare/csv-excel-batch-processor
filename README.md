## Features
- Batch processing
- Clean columns (trim whitespace)
- Sort columns
- Reorder columns
- Combine multiple files

## Usage
1. Select files or folder
2. Choose output folder
3. Select options
4. Run

## Notes
- If both Reorder and Sort are enabled, Reorder takes priority
